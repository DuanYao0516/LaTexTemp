# LaTeXTemp

自己做的中文课程报告LaTeX模板，包含封面、目录、摘要、正文、参考文献。

在 https://github.com/wxflogic/latex 基础之上修改而来。

示例见pdf文件。

可以直接上传到Overleaf上使用。

# 安装与编译

本模板需要使用Xelatex编译。推荐使用VS code + Texlive 环境。 

配置教程参考：https://zhuanlan.zhihu.com/p/38178015
